# Soma

```
Escreva uma função que receba um array com arrays
de números e retorne um array com a soma de todos os
elementos de cada array.
```

--
***Exemplo:***
```javascript
var entrada = [
    [2, 4, 3],
    [0, 7, 9],
    [23, 4, 11]
];

soma(entrada); // [9, 16, 38]

```
