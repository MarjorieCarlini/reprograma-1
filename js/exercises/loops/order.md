# Ordem

```
Escreva uma função que receba um array desordenado
de números e retorne um novo array ordenado.
```

--
***Exemplo:***
```javascript
var entrada = [9, 4, 7, 1, 2];

ordem(entrada); // [1,2,4,7,9]

```
