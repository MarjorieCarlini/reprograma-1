# Pirâmide

```
Escreva uma função que receba um número e escreva uma
pirâmide de asteriscos da altura desse número.
Cada nível da pirâmide deve ter um número ímpar 
de asteriscos.
```


***Exemplo:***
```javascript
pyramid(3);

  *
 ***
*****
```

```javascript
pyramid(4);

   *
  ***
 *****
*******
```
